FAME Workshop

Acara workshop yang bertemakan Futuristic Game Development (FAME) ini diselenggarakan oleh HIMTI Programming Class Division supaya mahasiswa dapat belajar membuat sebuah game futuristik dengan game engine Unity hanya dalam kurun waktu 30 menit.

Benefits

  • E-Certificate

    Anda akan memperoleh sebuah e-certificate yang menandakan kehadiran Anda dalam acara workshop ini.

  • Knowledge

    Anda akan memperoleh sekumpulan pengetahuan baru mengenai pembuatan game futuristik.

  • Experience

    Anda akan memperoleh pengalaman baru dengan belajar hal baru, terutama mengenai pembuatan game futuristik.
